# Phase 6 Sound Lab

Phase 6 upgrades Sound Lab into a diagnostic ear-training simulator.

## Documents

- [Sound Lab Blueprint](SOUND_LAB_BLUEPRINT.md): scenario model, interaction design, generated audio approach and future persistence plan.

## Phase 6 Outcome

The app now has:

- Expanded Sound Lab scenario data
- A richer `AudioExample` model for diagnosis, difficulty, A/B/C variants, meter profiles, first checks and recommended fixes
- Interactive A/B/C/reference playback using generated browser audio
- Scenario-driven waveform, spectrum and meter visuals
- Diagnosis and first-safe-check prompts
- Reveal/feedback behavior
- Related lesson and troubleshooting links

This phase still uses generated browser audio. Real church recordings, sourced clips and persisted attempt scoring should be added after the Digital Twin, asset registry and certification evidence flows mature.
