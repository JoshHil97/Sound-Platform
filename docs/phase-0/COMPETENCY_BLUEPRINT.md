# Sound Academy Competency Blueprint

## Competency Categories

### Knowledge Competencies

- Ministry purpose and communication priorities
- Sound fundamentals: frequency, amplitude, decibels, acoustics
- Signal flow from source to FOH and livestream
- X32 channel strip, buses, matrices, scenes and snippets
- Dante device, subscription, clocking and DVS concepts
- Logic channel strips, auxes, buses, plugins and output routing
- Waves plugin purposes
- Service SOPs and incident logging

### Practical Competencies

- Set gain on speech, vocal, DI and choir sources
- Patch and name channels
- Run line checks
- Build P16 and monitor sends
- Prepare wireless microphones
- Build Logic stream template paths
- Apply EQ/dynamics/plugin moves with a reason
- Capture screenshots/evidence

### Operational Competencies

- Prepare for Sunday service
- Prioritise speech and service flow
- Avoid panic scene recall
- Communicate with stage and mentors
- Make one change at a time
- Log deviations and handovers

### Troubleshooting Competencies

- Trace source to destination
- Identify shared path versus branch-specific faults
- Diagnose no mic, no stream, Dante issue, feedback, wireless dropout, P16 fault and Logic output fault
- Use X32 meters/PFL, Dante Controller and Logic meters
- Escalate safely

### Communication Competencies

- Explain actions in plain language
- Use calm talkback etiquette
- Communicate service risk clearly
- Write useful incident notes
- Ask musicians clear monitor questions

### Leadership Competencies

- Coach without overwhelming volunteers
- Assess consistently
- Approve evidence
- Maintain SOPs
- Govern change and renewal

## Academy Mapping

Foundations Academy:

- Knowledge: high
- Practical: introductory
- Operational: shadowed only
- Troubleshooting: first safe checks
- Communication: beginner vocabulary
- Leadership: not required

FOH Operator Academy:

- Knowledge: applied console knowledge
- Practical: high X32/P16/wireless practice
- Operational: supervised to independent service
- Troubleshooting: common live faults
- Communication: stage/team clarity
- Leadership: handover discipline

Livestream Academy:

- Knowledge: Dante, Logic, Waves and loudness
- Practical: high Logic/Dante workflow practice
- Operational: stream rehearsal and service operation
- Troubleshooting: stream-specific branches
- Communication: FOH/stream coordination
- Leadership: post-service review notes

Systems Academy:

- Knowledge: advanced routing/network architecture
- Practical: advanced X32/Dante/virtual soundcheck
- Operational: system recovery and rollback
- Troubleshooting: timed fault isolation
- Communication: incident leadership
- Leadership: documentation ownership

Leadership Academy:

- Knowledge: assessment and coaching standards
- Practical: mentor observations
- Operational: service debriefs
- Troubleshooting: escalation leadership
- Communication: feedback process
- Leadership: high

Technical Director Academy:

- Knowledge: governance and architecture
- Practical: approval workflows
- Operational: production readiness
- Troubleshooting: root-cause governance
- Communication: stakeholder reporting
- Leadership: highest
