# Phase 4 Academy Experience

Phase 4 turns the Academy area from a flat LMS-style module list into a specialist pathway experience.

## Documents

- [Academy Experience Blueprint](ACADEMY_EXPERIENCE_BLUEPRINT.md): pathway UI model, reusable components, route map and next implementation steps.

## Phase 4 Outcome

The Academy now has:

- A premium pathway hub at `/academy`
- Dedicated specialist path pages at `/academy/paths/[academySlug]`
- Reusable academy UI components for hero panels, sidebars, module cards, command panels, skill trees and certification bands
- Module pages that link back to their parent specialist path
- Data-driven integration with Phase 3 academies, modules, competencies, skills, skill trees and certifications

This phase intentionally builds framework and interaction structure first. Deeper lesson simulators, source-backed visual libraries, audio labs and mentor workflows belong to later phases.
